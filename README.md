# Todo App

A minimal full-stack todo app with a React frontend and Node.js + Express backend.

## Run backend

1. `cd backend`
2. `npm install`
3. `npm start`

The backend listens on `http://localhost:4000`.

## Run frontend

1. `cd frontend`
2. `npm install`
3. `npm run dev`

The frontend runs on `http://localhost:5173`.
